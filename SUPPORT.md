# Support — Revive (`rv`)

---

## Free Support (GitHub)

All bug reports, feature requests, and general questions are handled through GitHub:

- **Bug reports**: [Open an issue](https://github.com/0xkhdr/revive/issues/new?template=bug_report.md)
  — include your OS, Python version, `rv --version` output, and the full error traceback.
- **Feature requests**: [Open a discussion](https://github.com/0xkhdr/revive/discussions/new?category=ideas)
- **General questions**: [GitHub Discussions Q&A](https://github.com/0xkhdr/revive/discussions/new?category=q-a)

See [TROUBLESHOOTING.md](TROUBLESHOOTING.md) for common errors and fixes before opening a new issue.

## Security Vulnerabilities

Do **not** report security vulnerabilities in public GitHub issues.

Use [GitHub Private Vulnerability Reporting](https://github.com/0xkhdr/revive/security/advisories/new)
or see [SECURITY.md](SECURITY.md) for the full disclosure policy.

---

## Diagnostic Information

When filing a bug, include the output of:

```bash
rv --version
rv doctor --json
python3 --version
uname -a
```
