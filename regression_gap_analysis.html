§ 5

Architecture & Security Gap Register

Architecture Gaps

ID Category Severity Evidence & Remediation
A-003 Provider Coverage Collapse High Cargo, Nix, Dnf, Pacman and Pip managers average 24% testing coverage.
Fix: Create unified provider mock harnesses.
A-004 Restore Engine Gaps High Core restore.py runs at only 72% coverage. Core error paths remain completely untested.
Fix: Write explicit error path and rollback tests.
A-007 Broken Golden Path Medium 'rv clone' command listed in architecture plans, but remains completely unimplemented.
Fix: Expose clone utility in CLI controller.

Security Gaps

ID Category Severity Evidence & Remediation
S-001 Action Supply-Chain Risk Medium CI workflows utilize mutable tag references (@v4, @v5) which can be force-pushed.
Fix: Lock CI references to explicit, validated commit SHAs.
S-003 Localhost Token Leak Medium Authentication tokens run over raw HTTP. Vulnerable if bound to external network hosts.
Fix: Raise hard-crash error when binding non-loopback IPs without TLS.
S-004 Native Extensions Escape High Plugins using bundled native compiled binaries (.so files) bypass standard import interceptors.
Fix: Implement formal Linux seccomp profiles for containment.

Synthesis & Roadmap
