🛡️ Security Best Practices

  • Identity File: Keep your private key in a secure folder outside of git repositories.
  • Access Permissions: Revive automatically applies strict octal 0600 permissions to decrypted secret assets.
  • Key Rotation: Rotate active credentials routinely using the CLI rv secret rotate command.
